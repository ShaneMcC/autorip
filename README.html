This collection of scripts is designed to be used for automatic ripping of DVDs or CDs with little to no user intervention.

The script automatically detects when a disk has been inserted, and then decides what type of disk it is (Audio CD, or Video DVD) and starts the appropriate ripping process.

obtaining the code

The code for this project can be obtained using either of the following:

  • git clone git://git.soren.co.uk/shane/autorip.git/
  • git clone http://git.soren.co.uk/r/shane/autorip.git/

requirements

These scripts require a few packages for all the functionality
These can be installed (on ubuntu) by using or letting install.sh do it for you: 

sudo apt-get install hal mencoder abcde id3v2 lame lsdvd dvdbackup
dvdbackup is optional and can be used in place of lsdvd, or as a fallback incase lsdvd fails.

using the code

Once the code has been downloaded, edit diskinserted.conf.example to your liking then run install.sh to install the hooks to check for disks being inserted.
